# مستندات فارسی WhiteDNS

این پوشه مستندات فارسی WhiteDNS را نگه می‌دارد. هدف این مستندات این است که کاربر
بدون نیاز به حدس‌زدن، معنی تنظیمات و اثر آن‌ها روی اتصال، سرعت، پایداری، مصرف
حجم، باتری و امنیت را بفهمد.

## فایل‌ها

- [راهنمای کاربر](WHITE_DNS_USER_GUIDE.md): توضیح کامل مفاهیم، پارامترها، حالت
  Proxy، حالت VPN، Resolverها، Warmup، Diagnostics و اشتراک‌گذاری.
- [راهنمای عیب‌یابی](WHITE_DNS_TROUBLESHOOTING.md): مسیر مرحله‌ای برای مشکل‌هایی
  مثل وصل می‌شود ولی چیزی باز نمی‌شود، گیر کردن درصد، مصرف آپلود، قطع شدن در
  پس‌زمینه و خطاهای Resolver.
- [برنامه QA](WHITE_DNS_QA.md): سناریوهای تست دستی برای انتشار، امنیت، UI، اتصال،
  وارد کردن پروفایل، Resolver، VPN، Proxy و Diagnostics.

نسخه انگلیسی مستندات از [فهرست مستندات](../README.md) شروع می‌شود.

## اصول نوشتاری

- متن باید ساده، مستقیم و قابل اجرا باشد.
- هیچ کلید، رمز، لینک پروفایل واقعی، IP خصوصی کاربران یا لاگ حساس نباید در
  مستندات قرار بگیرد.
- هر پارامتر باید با اثر کاربرمحور توضیح داده شود: سرعت، پایداری، مصرف حجم،
  باتری، امنیت، یا ریسک در شبکه محلی.
- هر راهنمای عیب‌یابی باید یک ترتیب مشخص داشته باشد و کاربر را به تغییر همزمان
  چند تنظیم تشویق نکند.
